Udacity Reflection Part2

Grid Based Design

This is what we use to structure our webpages. 12 columns across means that we can have the most flexibility on how wide we want each column while keeping them big enough to hold text.

Responsive Webpages

We all use so many devises these days to access the internet and they can have widely different screen sizes. By using % instead of px we are able to make the content adapt to the different sized screens. By adding an @media formula we can also change the way content appears on smaller screens so that it is more usable as we will be using our fingers and not a mouse.

Semantic structure

We need to use tags that make sense to humans and machine. The title should go in a h1 tag because this needs to be the most prominent.

Other reflections

A couple of other interesting things to note from the tutorial is the css 'overflow: auto;' which means that you can limit the area that the text is taking and a scrollbar will apear. I also appreciated the mention of Google Fonts which look to be a fantastic resource and placepuppy.it which will hold the place of an image while the correct one is found.
