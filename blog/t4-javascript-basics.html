JavaScript compared to HTML and CSS

HTML is the content of the website, CSS is the layout/design and JavaScript makes all the moving parts possible.

Control Flow and Loops

JavaScript has Control Flow which is like getting dressed in the morning. We first evaluate what kind of cloths are needed, if casual or smart, winter or summer. All these decisions narrow down the outfit we will eventually leave don. Then we can issue the command to put on the appropriate attire, footwear and accessories.

Looping is when we need to do the same task many times like when we are doing the dishes. We scrub each dish or pot and then we put it down. We specify first how many items need to be washed so that we know when to finish. If we don't set limits then we can fall into what is called an Infinite Loop.

Accessing data from Arrays and Objects Literals

An Array is a list of items or numbers that go in sequence. They are always accessed with []. Object literals can be accessed with either a dot or brackets.

Functions and their uses

A function completes a task when it is called to do so. This can be written into the code and self-started or it can be started by the click of a user.

They are useful because they can be written many ways to produce different results.
