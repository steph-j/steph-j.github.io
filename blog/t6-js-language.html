JS Language

Sun 24Apr16

Parentheses ()

  1. EXECUTE A FUNCTION ie var result = doSomething()
  2. SURROUND A CONDITIONAL STATEMENT ie if (i < 0) {
  3. SPECIFY THE ORDER OF OPERATION ie ((1+1) * 2)

Brackets []

  1. CREATE ARRAYS ie myArray = [];
  2. ACCESS AN ELEMENT OF AN ARRAYS ie myArray[0]

Braces {}

  1. TO DEFINE AT OBJECT LITERAL
  2. ENCLOSE BLOCKS OF CODE ie everything inside a function and everything inside a conditional statement

Single quotes '' & Double quotes ""

These can be used interchangeably

  1. TO DEFINE A STING
  2. IF YOU WANTED SPEECHMARKS IN YOUR SENTENCE ie 'The man said "I am the man" and then he was done'
